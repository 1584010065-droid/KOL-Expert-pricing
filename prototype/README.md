# 达人核价可交互原型

## 打开方式
在浏览器直接打开：

`/Users/qiuzhidong/数说/26-4/KOL/4.21 kol达人核价模块/prototype/index.html`

或在项目目录执行：

```bash
python3 -m http.server 5173
```

然后访问 `http://localhost:5173/prototype/index.html`。

## 原型页面
- 用户搜索达人页面（参考上传筛选样式）
- 候选达人列表页
- 批量核价结果列表页（含 AI 异步回填模拟）
- 达人详情页（含基础信息、结论、AI分析、价格分布、趋势、风险、相似达人）

## 交互说明
- 搜索页点“搜索达人”进入候选列表
- 候选列表单选可进详情，多选可发起批量核价
- 批量核价页会逐条模拟“AI分析中 -> 已完成”
- 从批量页或候选页都可进入达人详情页
